Infrastructure provided by Deusto
=================================

Deusto provides the following computation environment:

    * 144 vCPU.
    * 375GB RAM.
    * Storage:

        * SSD: 7.04 TB
        * HDD: 19.8 TB
    * Powered by `Kubernetes <https://kubernetes.io/docs/concepts/overview/what-is-kubernetes/>`_.
